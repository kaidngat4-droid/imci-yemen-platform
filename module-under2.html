وزارة الصحة العامة والبيئة

🏥 منصة التدريب الطبي للرعاية التكاملية لصحة الطفل (IMCI)

👶 تقييم الطفل أقل من شهرين

بروتوكول IMCI - منظمة الصحة العالمية (WHO)

⚠️ الفئة العمرية: 0-59 يوم

🚨 الخطوة 1: علامات الخطر العامة

تحقق من وجود أي من العلامات التالية (إذا وجدت ← احال فوراً):

  • ✓ عدم القدرة على الرضاعة أو الشرب
  • ✓ نوبات صرع / اختلاجات
  • ✓ غيبوبة أو فقدان الوعي / خمول شديد
  • ✓ استفراغ كل شيء (Everything)
  • ✓ صعوبة شديدة في التنفس / زرقة
  • ✓ حمى شديدة (> 38°C) أو انخفاض حرارة (< 35.5°C)

🏥 احال فوراً للمستشفى!

وجود أي علامة خطر عامة يستوجب الإحالة الطارئة

أثناء الإحالة: أعطِ المضاد الحيوي + حافظ على دفء الطفل

🦠 الخطوة 2: تقييم العدوى البكتيرية الخطيرة

  • ✓ احمرار أو خروج صديد من السرة
  • ✓ احمرار الجلد مع بثور أو خروج صديد (عدوى جلدية)
  • ✓ درجة حرارة ≥ 37.5°C (إبطي) أو ≥ 38°C (شرجي)
  • ✓ انخفاض حرارة < 35.5°C
  • ✓ حركات أقل من الطبيعي أو تهيج غير طبيعي

💧 الخطوة 3: تقييم الإسهال والجفاف

علامات الجفاف:

  • ✓ جلد بطيء العودة (> 2 ثانية) - علامة جفاف شديد
  • ✓ عيون غائرة
  • ✓ جفاف الفم واللسان
  • ✓ نقص عدد مرات التبول (أقل من 4 مرات/يوم)

💊 خطة العلاج:

الجفاف الشديد: احال فوراً + ORS أثناء الإحالة

جفاف بعض: ORS 75ml/kg خلال 4 ساعات + متابعة الرضاعة

لا جفاف: ORS 50ml بعد كل برازة + استمرار الرضاعة الطبيعية

🍼 الخطوة 4: تقييم التغذية والرضاعة

  • ✓ هل يرضع الطفل رضاعة طبيعية حصرية؟
  • ✓ عدد مرات الرضاعة: 8-12 مرة/يوم
  • ✓ وضعية الرضاعة صحيحة (الطفل ملتصق جيداً بالثدي)
  • ✓ وزن الطفل يزداد بشكل طبيعي (20-30g/يوم)
